# mrfaiergm_studios.restartCommand_MrFaiergm_SRVPLG
плагин для управления перезапуском сервера.

## Особенности
- команда для запуска перезапуска `/prestart`.
- Команда для отмены перезапуска `/pstop`.
- команда для перезагрузки конфига `/prestart reload`
- - команда для помощи  `/Phelp`
- Гибкая настройка через `config.yml`.
- Оповещение Администрации при перезапуске.

## Установка
1. Скачайте файл плагина (.jar).
2. Поместите его в папку `plugins` вашего сервера.
3. Перезапустите сервер.
